Description: Proyecto con el fin de llevar un sistema de inventario de los productos de limpieza del INA de Upala por el cual se llevaba cuanta cantidad se tenía y cuanto se debería pedir para el próximo pedido, esta información era traída de un Excel hacia una tabla en base de datos para poder administrar bien la información recibida.
